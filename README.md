## 📖 Introduction

<details>
<summary>✨ Key Points (Click to Expand)</summary>

| Topic | Details |
|-------|---------|
| **AI-Assisted** | Narrow tasks like code completion with existing rituals slowing velocity |
| **AI-Managed** | Single-shot autonomous development with minimal human involvement |
| **Limitations** | Both approaches have limits in transformative benefits |
| **AI-DLC Goal** | Balances AI’s heavy lifting with human oversight |
| **Phases** | Inception • Construction • Operations |
| **Workflow** | `Task → Plan → Clarify → Refine → Approve → Execute → Verify` |
| **Collaboration** | Cross-functional collaboration is essential |

## 📖 Workflow & Collaboration

<details>
<summary>✨ Key Points (Click to Expand)</summary>

- Workflow: `Task → Plan → Clarify → Refine → Approve → Execute → Verify`  
- Break problems, share responsibility, disagree & commit  
- One-way door decisions need deliberation  
- Five critical points: clarification, plan approval, execution, quality, iteration vs completion  
- Human collaboration at AI speed prevents bottlenecks
